\doxysubsection{Detailed Description}
Implementation of a reference counted string This string is lighweight with the string class itself being little more than a slightly smart pointer.

Primarily created to satisfy a need I have for dynamic invariant strings in a re2cc/lemon based project which will be passing a ridiculous number of strings around.
